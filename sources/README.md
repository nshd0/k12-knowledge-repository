# sources/

**Purpose:** Source index, citation records, and official reference links used to verify and trace all knowledge entries.

Every entry in `data/source_index.csv` has a corresponding record here. This folder provides traceability between knowledge summaries and their upstream official documents.

## Rules

- Only link to official sources: MoE, NCERT, CBSE, NCTE, state DoE, or DIKSHA
- All source URLs must be publicly accessible and stable
- Do not store copies of source documents — link only
