# Weekly Update Checklist

## Every Monday
- [ ] Check CBSE SARAS: https://saras.cbse.gov.in/saras/Home/Circulars
- [ ] Check CBSE student circulars: https://www.cbse.gov.in
- [ ] Check Delhi DoE: https://www.edudel.nic.in
- [ ] Check DIKSHA: https://diksha.gov.in
- [ ] Run `python scripts/ingest_sources.py`
- [ ] Run `python scripts/tag_documents.py`
- [ ] Run `python scripts/weekly_digest.py`
- [ ] Add new items to relevant CSV files
- [ ] Update `docs/update_log.md`
- [ ] Push changes to GitHub

## Every Month
- [ ] Check Ministry of Education: https://www.education.gov.in
- [ ] Check NCERT: https://ncert.nic.in
- [ ] Check CBSE Academic: https://cbseacademic.nic.in
- [ ] Check PARAKH: https://parakh.gov.in
- [ ] Check NIPUN Bharat: https://nipunbharat.education.gov.in
- [ ] Check NCTE: https://ncte.gov.in
- [ ] Review and archive superseded items
- [ ] Update source index verification dates
