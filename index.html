Core message

The repository is not just storage. It is a governed retrieval layer that lets AI agents deliver consistent, context-aware answers without drifting across versions or teams.

Why this framing works

A shared knowledge layer is a natural fit for K–12 AI systems because schools need role-specific assistance built on consistent institutional context for lesson planning, communication, compliance, and operational guidance.

Consistency

All agents retrieve from the same approved sources, reducing answer drift and policy conflicts across the school.

Ownership

The school or organisation controls the canon, the update cycle, and the allowed source set — not the AI vendor.

Usability

Teachers, principals, parents, and students each see a tailored experience without fragmenting the underlying truth.

Recommended product structure

One master repository plus stakeholder-specific notebooks or views. GitHub as system of record. NotebookLM as the interaction layer.

Master repository

Contains governed source files, summaries, update logs, compliance rules, and exports that feed all downstream experiences.

Agent architecture

A governed retrieval pipeline rather than a free-form chatbot. The agent reads from canonical content, respects content boundaries, and surfaces citations for every claim.

1
Ingest

Approved documents are exported from GitHub into clean, source-tagged files on a weekly cadence via GitHub Actions.

2
Index

Metadata — stakeholder, grade band, topic, source, effective date, and content owner — improves retrieval quality and agent accuracy.

3
Answer

The agent composes responses from the same canonical source set for every stakeholder experience, keeping answers consistent.

4
Audit

Versioning, update logs, and attribution headers preserve full transparency for editors, reviewers, and compliance checks.

Curriculum domain

Learning outcomes, scope and sequence, pedagogy guides, and subject resources organised by grade band.

Policy domain

NEP 2020 summaries, NCF-SE 2023 key points, CBSE circulars, official guidelines, and compliance notes.

Communication domain

Parent notices, school calendar updates, announcements, and simplified stakeholder-facing summaries.

Governance and attribution

The knowledge layer stays trustworthy only if every file carries explicit origin, ownership, and licence information.
