# Stakeholder Guide

## Principal / School Leader
- Review circulars from CBSE and Delhi DoE weekly.
- Track NEP 2020 compliance action items.
- Monitor school affiliation updates.

## Teacher
- Use DIKSHA and NISHTHA for capacity building.
- Access NCERT textbooks and CBSE sample papers for lesson planning.
- Apply NCF 2023 learning outcomes in classroom delivery.

## Student
- Download NCERT textbooks for all subjects.
- Practice with CBSE sample papers and marking schemes.
- Use DIKSHA for interactive learning content.

## Parent
- Follow CBSE exam date sheet and results notifications.
- Read Delhi DoE admission notifications.
- Track school circulars for important dates.

## Coordinator
- Maintain source index and verification log.
- Run weekly_digest.py every Monday.
- Add new circulars to circulars.csv immediately upon release.
