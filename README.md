## 🎯 Purpose

This repository **aggregates and organizes** publicly available Indian K-12 educational content from official government sources. It serves as:

- 🔍 **Centralized Discovery**: One-stop access to curriculum, policy, and assessment resources
- 🤖 **AI-Ready Infrastructure**: Structured data for RAG systems and educational AI applications  
- 📊 **Research Platform**: Analyzed, catalogued data for education technology research
- 🎓 **Stakeholder Resource**: Direct access for educators, students, policy makers, and parents

> **Note**: This repository does **not** create or claim ownership of source materials. All content originates from official Government of India education platforms. See [Copyright & Attribution](#-copyright--attribution) for details.

## 🔍 How to Use This Repository

### Quick Search

**By Grade**: Browse [`knowledge/lessons/by_grade/grade_XX/`](knowledge/lessons/)  
**By Subject**: Navigate [`knowledge/curriculum/cbse/[subject]/`](knowledge/curriculum/)  
**By Keyword**: Use GitHub search or open [`catalog.csv`](knowledge/metadata/catalog.csv) in Excel

### Download Specific Content

1. Navigate to desired folder (e.g., `knowledge/curriculum/ncert/class_10/mathematics/`)
2. Click on file to view
3. Download raw file or clone entire repository

### Access Data Catalog

The complete, searchable catalog of all documents:
```bash
cat knowledge/metadata/catalog.csv
```

Or open in Excel/Google Sheets: [catalog.csv](knowledge/metadata/catalog.csv)

### Search by Metadata

Catalog includes 24 fields per document:
- Source, Grade, Subject, Topic
- File paths, URLs, dates
- Quality scores, keywords
- Learning outcomes, difficulty levels

## 🔄 Data Updates

### Automated Scraping

- **Schedule**: Every Sunday at 2:00 AM IST
- **Method**: GitHub Actions workflow
- **Process**:
  1. Fetch latest content from 8 official sources
  2. Validate data quality and integrity
  3. Update catalog with new entries
  4. Commit changes to repository

**View workflow**: [`.github/workflows/run-data-scraping.yml`](.github/workflows/run-data-scraping.yml)  
**Scraping guide**: [`scraper/SCRAPING_GUIDE.md`](scraper/SCRAPING_GUIDE.md)

### Data Quality

- ✓ Source verification (official .gov.in/.nic.in domains)
- ✓ Integrity checks (MD5/SHA checksums)
- ✓ Metadata validation (automated)
- ✓ Duplicate detection
- ✓ Quality scoring (0.0-1.0 scale)

## 👥 Contributing

### Report Issues

- **Missing Content**: [Open an issue](https://github.com/nshd0/k12-knowledge-repository/issues) with source details

### Suggest New Sources

**Requirements:**
1. Must be official Government of India education platform
2. Publicly accessible without authentication  
3. Provides structured, machine-readable content
