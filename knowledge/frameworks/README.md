# K12 Education Frameworks Repository

## Overview
This directory contains comprehensive documentation of all major education frameworks applicable to K-12 education in India. These frameworks guide curriculum development, pedagogy, assessment, and educational policy implementation.

## Available Frameworks

### ✅ Completed

#### 1. National Education Policy (NEP) 2020
**File**: `nep_2020_framework.md`  
**Status**: ✅ Complete  
**Description**: India's comprehensive education reform framework replacing NPE 1986. Includes the 5+3+3+4 pedagogical structure, competency-based assessment, holistic development, and technology integration.

**Key Topics Covered**:
- Vision and principles
- 5+3+3+4 pedagogical structure
- Assessment reforms
- Curriculum and pedagogy guidelines
- Teacher education
- Technology in education
- Equity and inclusion measures
- Implementation timeline

## Framework Categories

### By Scope
1. **National Frameworks**: NEP 2020, NCF 2023, PARAKH
2. **Board-Specific**: CBSE, State Boards
3. **Stage-Specific**: ECCE, FLN, Secondary
4. **Domain-Specific**: Digital Education, Vocational, Special Education
5. **Competency-Based**: 21st Century Skills, SEL

### By Implementation Authority
1. **Ministry of Education (MoE)**: NEP 2020, NCF 2023
2. **NCERT**: Curriculum frameworks, textbooks
3. **CBSE**: Board examinations, assessment
4. **NCTE**: Teacher education
5. **State Governments**: State boards, SCERT
