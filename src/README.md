# src/

**Purpose:** Frontend source code for the GitHub Pages knowledge hub, built with Vite.

This folder contains the React/JS components, pages, and styles that render the public-facing knowledge site.

## Rules

- Follow the existing code style and component structure
- All changes must pass the Vite build check in `ci-cd.yml` before merging
- Do not put knowledge content or data files here — those belong in `knowledge/` and `data/`
