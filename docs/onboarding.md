# Onboarding Guide

## Who is this repository for?
School leaders, teachers, students, parents, and coordinators working in K-12 education in India.

## How to use it
1. Browse by category using the web app.
2. Search by topic, board, subject, or circular.
3. Click "Open source" to access the official document.
4. Check the weekly digest for new updates.

## How to contribute
1. Add new rows to the relevant CSV in `data/`.
2. Follow the column schema in each file.
3. Push to `main` — Netlify will redeploy automatically.

## Update responsibilities
| Role | Frequency | Sources |
|------|-----------|----------|
| Repository admin | Weekly | CBSE, DoE, DIKSHA |
| Repository admin | Monthly | MoE, NCERT, PARAKH |
